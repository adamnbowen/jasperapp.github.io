Overview

Jasper is a tool for reading issues and pull requests in GitHub efficiently. You will be able to read issues sorting author, repository, organization, and more.

Your Issues

By default, Jasper automatically shows issues - ones you are involved in, your team is mentioned in, and of repositories you watch. You'll never miss issues.

Stream

Jasper supports streaming - show issues in a particular repository, of a particular user, with a particular word, etc. You can read active issues in GitHub.

Filter

Jasper supports powerful filtering - select unread issues, issues with a particular label, unassigned issues, etc. You can easily and quickly find what you want to check out now.

Sort

Jasper supports powerful sorting - rearrange the order of issues by update date, read date, due date, etc. You can read the issues you're most interested in.
